# Resume Skill Matcher

A simple **Streamlit web application** that compares a resume with a job description and calculates a **skill match percentage** using **TF-IDF** and **cosine similarity**.  
It also highlights **matched skills** and **missing skills** to help improve resumes.

---

## Features
- Paste resume text and job description
- Calculates skill match percentage
- Identifies matched and missing skills
- Clean and interactive UI built with Streamlit
- Beginner-friendly project

---

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity
- Regular Expressions (re)

## Installation & Run Locally


### 1️⃣ Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/Resume-Skill-Matcher.git
```

### 2️⃣ Navigate to the project folder
```bash
cd Resume-Skill-Matcher
```

### 3️⃣ Install dependencies
```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Streamlit app
```bash
streamlit run app.py
```
